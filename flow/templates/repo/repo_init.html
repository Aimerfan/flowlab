Command line 說明
你可以使用以下指令，從你的電腦上傳現有的文件。
Git 全局設置

git config --global user.name "{{ user.username }}"
git config --global user.email "{{ request.user.email }}"
建立一個新的儲存庫

git clone {{ gitlab_http_url }}
cd {{ info.name }}
git switch -c main
touch README.md
git add README.md
git commit -m "add README"
git push -u origin main
Push 一個已存在的資料夾

cd existing_folder
git init --initial-branch=main
git remote add origin {{ gitlab_http_url }}
git add .
git commit -m "Initial commit"
git push -u origin main
Push 一個已存在的 Git 儲存庫

cd existing_repo
git remote rename origin old-origin
git remote add origin {{ gitlab_http_url }}
git push -u origin --all
git push -u origin --tags
